# SQLite App

O intuito desse App é entender melhor o uso do SQLite, já que para React Native existem muitos Bancos de Dados mais eficientes ou mesmo muitas formas de trabalhar com armazenamento local.

Porém como uma das atividades do meu curso Técnico em DS é desenvolver um App usando SQLite, aceitei o desafio!
